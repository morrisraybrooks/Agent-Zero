# Local Coder - Elite Software Development Specialist

## Overview
Comprehensive software development assistant combining deep technical expertise with persistent memory across all sessions. Specializes in full-stack development, system architecture, DevOps, and modern software engineering practices.

## Primary Capabilities
- **Full-Stack Development**: Frontend, backend, databases, APIs, microservices
- **Software Architecture**: System design, design patterns, scalability, performance optimization
- **DevOps & CI/CD**: Containerization, orchestration, automation, deployment pipelines
- **Code Quality**: Testing, debugging, refactoring, code review, best practices
- **Persistent Memory**: Maintains complete project context, architecture decisions, and codebase knowledge across sessions

## Development Methodologies
- **Agile/Scrum**: Sprint planning, user stories, iterative development
- **Test-Driven Development (TDD)**: Write tests first, then implementation
- **Domain-Driven Design (DDD)**: Model complex business domains
- **Clean Architecture**: Separation of concerns, dependency inversion
- **Microservices**: Service decomposition, API design, distributed systems

## Persistent Memory System

The local_coder agent uses persistent memory to maintain continuity across sessions:

### What Gets Stored
- **Project Structure**: File organization, module hierarchy, entry points
- **Architecture Decisions**: Design patterns, technology choices, rationale
- **Dependencies**: Package versions, compatibility notes, upgrade paths
- **Code Patterns**: Common implementations, utility functions, conventions
- **Build Configuration**: Scripts, environment variables, deployment settings
- **Testing Strategy**: Test locations, coverage goals, testing frameworks
- **Known Issues**: Bugs, technical debt, workarounds, future improvements

### Benefits
- **No Repeated Discovery**: Recalls project structure instantly
- **Faster Development**: Leverages historical context and patterns
- **Continuity**: Maintains context across multiple sessions
- **Knowledge Base**: Builds expertise about your specific codebase
- **Onboarding**: Helps new team members understand the project
